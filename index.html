Laboratory Work 1: Introduction to Git and Version Control

The primary objective of this laboratory work is to introduce students to the concepts and tools of version control, with a focus on Git. By the end of this lab, students will understand how to use Git for managing code versions and collaborating with others in software projects. They will learn how to track changes, create branches, merge code, and resolve conflicts in a collaborative development environment.
